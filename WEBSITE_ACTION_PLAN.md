# Website Action Plan

This checklist is a practical roadmap for improving, launching, and growing this website.

## 1) Immediate fixes (this week)

- [ ] Confirm every page has unique `<title>` and meta description tags.
- [ ] Verify all navigation links work on desktop and mobile.
- [ ] Check every image has descriptive `alt` text.
- [ ] Add/update a clear CTA (quote form, phone, and WhatsApp if used) on every service page.
- [ ] Test contact paths end-to-end (phone tap, email tap, form submit).
- [ ] Ensure favicon and logo load correctly across all pages.

## 2) Trust and conversion improvements

- [ ] Add a dedicated contact form page if one does not exist.
- [ ] Add before/after sections with project details (location, budget band, timeline).
- [ ] Add stronger social proof: customer reviews with dates and service type.
- [ ] Add service-area coverage block (towns/cities served).
- [ ] Add guarantees/credentials (insurance, years of experience, certifications).
- [ ] Ensure every service page has one primary conversion goal (call, form, quote).

## 3) SEO foundations

- [ ] Add LocalBusiness schema (JSON-LD) to key pages.
- [ ] Add Breadcrumb schema on inner pages.
- [ ] Verify `sitemap.xml` includes all important URLs and submit to Search Console.
- [ ] Add/validate `robots.txt`.
- [ ] Make URL slugs consistent and human-readable.
- [ ] Add internal links between related services and gallery pages.

## 4) Performance and technical quality

- [ ] Convert large images to modern formats (WebP/AVIF) where possible.
- [ ] Add lazy-loading to non-critical images.
- [ ] Serve minified CSS/JS in production.
- [ ] Enable long cache headers for static assets.
- [ ] Run Lighthouse and target 90+ for Performance, SEO, and Best Practices.
- [ ] Validate mobile responsiveness for common breakpoints (320, 375, 768, 1024+).

## 5) Analytics and lead tracking

- [ ] Install GA4 and verify pageview events.
- [ ] Configure conversion events (phone click, form submit, quote button).
- [ ] Connect Google Search Console and Bing Webmaster Tools.
- [ ] Add UTM tracking to paid/social campaign links.
- [ ] Create a simple weekly KPI sheet (traffic, leads, conversion rate).

## 6) Content growth plan

- [ ] Build out location pages for high-priority service areas.
- [ ] Publish at least 1 case-study style project per month.
- [ ] Create FAQ content around pricing, timelines, and maintenance.
- [ ] Add seasonal pages/offers (spring landscaping, winter maintenance, etc.).
- [ ] Refresh gallery monthly with latest completed jobs.

## 7) Operations and maintenance

- [ ] Document deployment and rollback steps for fast updates.
- [ ] Set a monthly broken-link and QA check.
- [ ] Back up site files before major content updates.
- [ ] Keep image naming and page naming conventions consistent.
- [ ] Review and update old testimonials/content every quarter.

---

## Suggested execution order

1. Conversion + contact path reliability.
2. Technical SEO basics (metadata, schema, sitemap, robots).
3. Performance image optimization and caching.
4. Analytics/event tracking setup.
5. Ongoing content and local SEO expansion.

## Definition of “launch-ready”

- Every core page has working CTA paths.
- Tracking is installed and conversions are measured.
- Sitemap + robots are validated and submitted.
- Lighthouse scores are acceptable and mobile UX is solid.
- At least 3 recent testimonials and 6+ strong project photos are visible.
